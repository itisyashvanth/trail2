# PiggyPath

PiggyPath is a modern, vibrant, Duolingo-style web application designed to gamify financial literacy and personal finance management. Built with React and Vite.

## Features
- **Duolingo-inspired Interface:** Clean, minimalistic, and highly visual dashboard and profile.
- **Vibrant Pixel Aesthetic:** Neon colors (lime green, magenta, cyan) combined with deep dark mode backgrounds and custom pixel art components.
- **Gamified Progression:** Earn XP, complete daily quests, maintain streaks, and climb the leaderboard leagues.
- **Avatar Customization:** A comprehensive pixel art avatar creator.

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```

## Technologies Used
- React 18
- Vite
- Framer Motion
- Tailwind CSS
- Lucide React
